# Swift Algorithms & Logic Building 🧠

A repository dedicated to sharpening my problem-solving skills and application logic using strictly **Swift**. 

Rather than focusing on competitive programming, this repo is tailored toward industry-standard patterns (inspired by the LeetCode 75 / NeetCode 150) that directly translate to efficient iOS app development—such as parsing JSON with Hash Maps (Dictionaries) and managing memory with Two Pointers.

## 🗂 Focus Areas
1. **Arrays & Hashing:** Mastering Swift `Dictionary` and `Set` for $O(1)$ lookups.
2. **Two Pointers:** Optimizing memory and array traversal without nested loops.
3. **Sliding Window:** Efficiently calculating subarrays and dynamic data streams.
